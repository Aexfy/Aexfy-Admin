# Contribucion

Proceso (ISO 9001)
- Todos los cambios via pull request.
- Revision obligatoria y evidencia de pruebas.
- Actualizar `docs/change-log.md` y `docs/release-notes.md`.
- Registrar no conformidades en `docs/nonconformity-log.md`.

Checklist de PR
- [ ] Requisitos actualizados si aplica.
- [ ] Pruebas ejecutadas (unit/integracion/e2e si aplica).
- [ ] Evidencia adjunta (logs o capturas).
